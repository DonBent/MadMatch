# Frontend

React application for MadMatch tilbudsoversigt.

## Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on port 3000 by default.

## Features

- Display 15 mock tilbud
- Filter by butik (store)
- Filter by kategori (category)
- Responsive design (mobile + desktop)
- Shows: navn, butik, normalpris, tilbudspris, rabat%

## Testing

```bash
npm test
```

## Build

```bash
npm run build
```
